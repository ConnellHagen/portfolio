Word Generator

I was inspired to make this by a similar project that my friend Ryan did earlier this year. The basic idea is that it takes in a text, and then determines what the probability is for a letter to be preceded by a letter. Using the concept of a Markov Chain, I then used the probabilites to generate faux english words. Sometimes they coincidentally generate real words, and sometimes they are absolute nonsense. I am working on improving the system to make more "realistic" words.

Practice Classes

These are some simple practice problem-style classes written for practice in java. Works like these were a majority of the work that I completed in my AP Computer Science A course in 2020-2021, and here are a few that I've completed in the following year. The current classes contained in this repository include: ScannerChopper, RegularPolygon and PermutationList. ScannerChopper was practice for using Java's Scanner class to chop up Strings. RegularPolygon and PermutationList challenged me to be able to take more abstract math concepts and write them in simple steps in code.
